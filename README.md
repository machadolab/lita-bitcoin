# lita-bitcoin

**lita-bitcoin** is a handler for [Lita](https://github.com/jimmycuadra/lita) that checks the current exchange rate of BTC/USD.

## Installation

Add lita-bitcoin to your Lita instance's Gemfile:

``` ruby
gem "lita-bitcoin"
```

## Usage

```
Lita: btc
```
